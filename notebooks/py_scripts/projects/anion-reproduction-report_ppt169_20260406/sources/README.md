# Reproduction: Anion Optimization for Bifunctional Surface Passivation in Perovskite Solar Cells

Reproduction of the machine learning workflow from:

> Xu, J., Chen, H., Grater, L. et al. Anion optimization for bifunctional surface passivation in perovskite solar cells. *Nat. Mater.* 22, 1507–1514 (2023).
> https://doi.org/10.1038/s41563-023-01705-y

## Overview

This repository reproduces the physics-informed ML pipeline that screens pseudo-halide (PH) anions for perovskite solar cell surface passivation. The original paper used DFT calculations to compute binding energies (Eb) for 267 PH anions on FA₀.₇₅MA₀.₂₅PbI₃ surfaces, then trained classification models to identify features that predict strong binding.

## ML Workflow

| Step | Description |
|------|-------------|
| **Dataset** | 267 PH anions, 19 molecular features, binary target (high/low Eb, threshold 3 eV) |
| **Round 1** | Remove 2 highly correlated features (MPI, La) → 17 features, train 5 models |
| **Feature Selection** | Identify top 4 features via RF importance + LR coefficients |
| **Round 2** | Retrain with 4 features (num_O, TPSA, HBA, HOMO) |
| **Target** | ROC AUC ≈ 0.87 ± 0.06, Accuracy ≈ 0.84 (Random Forest) |

## Models

1. Random Forest
2. Gradient Boosting Classifier
3. XGBoost
4. Logistic Regression (with L2 regularization)
5. Support Vector Classifier (RBF kernel)

## Key Findings (Reproduced)

PH anions with more oxygen atoms, larger TPSA, more H-bond acceptors, and lower HOMO levels exhibit stronger binding to the perovskite surface.

- Classification threshold: 3 eV
